# フェミニズム

## Overview

フェミニズムの思想的系譜・運動史・現代的論点を横断的に探求する。第一波から第四波までの歴史的変遷、交差性（インターセクショナリティ）・エコフェミニズム・リベラルフェミニズムなど多様な潮流の比較、そして日本における受容と固有の文脈（専業主婦規範・選択的夫婦別姓・性暴力対応など）を中心に扱う。「フェミニズムとは何か」を問い直すことで、自分自身の価値観や社会の構造を批判的に見る眼を養う。

## Key Questions

### 思想・歴史
- フェミニズムの第一波〜第四波とは何か。それぞれの時代背景・中心的主張・限界は？
- リベラルフェミニズム・ラディカルフェミニズム・社会主義フェミニズムの核心的な違いは何か？
- 交差性（インターセクショナリティ）とは何か。なぜクレンショーがこの概念を提唱したのか？

### 運動と実践
- #MeToo運動はフェミニズムの歴史においてどんな位置づけにあるか？
- フェミニズム運動内部での対立（TERFとトランスインクルーシブ派の論争など）はなぜ起きるのか？
- 「アライ（ally）」であることはどういう意味か。男性がフェミニズムに関わる際の留意点は？

### 日本の文脈
- 日本社会におけるフェミニズムの受容史——なぜ「フェミ」が蔑称化したのか？
- 選択的夫婦別姓・ジェンダーギャップ指数・性暴力法改正など、日本固有の課題の構造は？
- 上野千鶴子・田嶋陽子・伊藤詩織など、日本の代表的な論者が問うていることは何か？

### 批判的問い
- フェミニズムへの反論（反フェミ・エッセンシャリズム）の論拠は何か。どこが論点のズレになっているか？
- フェミニズムは「全女性を代表するか」という批判にどう応答しているか？
