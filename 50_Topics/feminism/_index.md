# フェミニズム

## Overview

フェミニズムの思想的系譜・運動史・現代的論点を横断的に探求する。第一波から第四波までの歴史的変遷、交差性（インターセクショナリティ）・エコフェミニズム・リベラルフェミニズムなど多様な潮流の比較、そして日本における受容と固有の文脈（専業主婦規範・選択的夫婦別姓・性暴力対応など）を中心に扱う。「フェミニズムとは何か」を問い直すことで、自分自身の価値観や社会の構造を批判的に見る眼を養う。

## Key Questions

### 思想・歴史
- フェミニズムの第一波〜第四波とは何か。それぞれの時代背景・中心的主張・限界は？
- リベラルフェミニズム・ラディカルフェミニズム・社会主義フェミニズムの核心的な違いは何か？
- 交差性（インターセクショナリティ）とは何か。なぜクレンショーがこの概念を提唱したのか？

### 運動と実践
- #MeToo運動はフェミニズムの歴史においてどんな位置づけにあるか？
- フェミニズム運動内部での対立（TERFとトランスインクルーシブ派の論争など）はなぜ起きるのか？
- 「アライ（ally）」であることはどういう意味か。男性がフェミニズムに関わる際の留意点は？

### 日本の文脈
- 日本社会におけるフェミニズムの受容史——なぜ「フェミ」が蔑称化したのか？
- 選択的夫婦別姓・ジェンダーギャップ指数・性暴力法改正など、日本固有の課題の構造は？
- 上野千鶴子・田嶋陽子・伊藤詩織など、日本の代表的な論者が問うていることは何か？

### 批判的問い
- フェミニズムへの反論（反フェミ・エッセンシャリズム）の論拠は何か。どこが論点のズレになっているか？
- フェミニズムは「全女性を代表するか」という批判にどう応答しているか？

## Findings

- フェミニズムは第一波（参政権・法的平等）→第二波（私的領域の政治化）→第三波（多様性・交差性）→第四波（デジタルアクティビズム）の四波で展開してきたが、各波は断絶ではなく前波の課題を引き継いでいる
- 「女性への抑圧の根源は何か」という問いに対し、リベラル（制度的不平等）・ラディカル（家父長制）・社会主義（資本制×家父長制）・交差性（複合的構造）という異なる診断がある
- 交差性（インターセクショナリティ）は1989年キンバリー・クレンショーが提唱。黒人女性の経験が「人種差別」でも「性差別」でも単独では説明できないという実践的問題意識から生まれた概念で、「分断のためではなく連帯のためのレンズ」
- #MeTooには「二つの誕生」がある——2006年のタラナ・バークによる黒人貧困層への支援活動と、2017年のハーヴェイ・ワインスタイン報道を契機とした世界的拡散。この「ずれ」自体が交差性の問題を体現している
- TERF（Trans-Exclusionary Radical Feminist）論争は表面上はトランス権利問題だが、根底には「女性というカテゴリの定義」と本質主義/構築主義の哲学的対立がある
- 日本の「フェミ」蔑称化は、1999年男女共同参画基本法施行後のジェンダー・バックラッシュ（自民党が主要拠点）、田嶋陽子のテレビ的矮小化、SNSの炎上構造の複合から生まれた
- 日本のジェンダーギャップ指数2025年版は148カ国中118位（G7最下位）。特に政治分野125位・経済分野112位が深刻。管理職女性比率は14.8%（127位）
- 2023年刑法改正により「不同意性交等罪」が創設。処罰根拠が「暴行・脅迫」から「同意の欠如」へ転換。明治以来110年ぶりの大改正
- 日本は民法750条により世界でただ一国、婚姻時の同一姓を強制する国
- 上野千鶴子の「資本制×家父長制の複合構造」テーゼは、日本の文脈でも有効な分析枠組みを提供している
- フェミニズムへの批判（「自然な差異」論・「もう不要」論・「逆差別」論）の多くは、フェミニズムの実際の主張を正確に把握していない論点のズレがある
- フェミニズムの内部批判（白人フェミニズム批判・帝国主義的フェミニズム批判）はフェミニズムを弱体化させるのでなく、より包括的にする自己更新の力
