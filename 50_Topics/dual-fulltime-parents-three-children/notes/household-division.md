# 家事・育児の分担実態

## 分担の現状（ソニー生命 2025年調査）

### 家事分担
| 担当 | 割合 |
|------|------|
| 主に妻 | 52.6%（子あり世帯では56.7%に増加） |
| 夫婦平等 | 25.0% |
| 主に夫 | 15.4% |

### 育児分担（子あり世帯 689名）
| 担当 | 割合 |
|------|------|
| 主に妻 | 51.8% |
| 夫婦平等 | 33.7% |
| 主に夫 | 7.5% |

## 時間の絶対量ギャップ

6歳未満の子どもがいる共働き世帯：
- 妻の家事時間：週391分（平均）
- 夫の家事時間：週114分（平均）
- **妻は夫の約3.4倍の時間を家事に費やしている**

## 夫婦の認識ギャップ

- 夫の63.4%が「自分がもっと担当すべき」と認識 → しかし実態は変わらない
- 妻の52.4%が「配偶者にもっと分担してほしい」と希望
- 育児でも：夫64.7%が「もっと担当すべき」、妻58.0%が「もっと分担してほしい」

→ 「わかってはいる、でも変わらない」という構造的問題

## ワンオペ化しやすいタスク

以下の理由でワンオペ化しやすい：

1. **可視化されにくいタスク**：病院予約・学校対応・保険更新・行事準備など
2. **即時対応が必要なタスク**：子どもの発熱、急な学校連絡
3. **細かい段取りが必要なタスク**：食事の献立・食材管理・子どもの持ち物管理

## 3人目が加わる時の複雑度増加

- 年齢差による生活リズムの違い（乳幼児 + 幼児 + 小学生が同居するケース）
- 保育園・幼稚園・小学校の行事が重なる
- 習い事の送迎、宿題サポート、学校対応が並走
- 兄弟間のケンカ調停・情緒サポートが加わる

## 負担軽減策（実践されている工夫）

1. まとめ買い：31.5%
2. 冷凍食品活用：27.0%
3. 外食・テイクアウト：22.0%
4. 朝夜のタスクを交代制にする（属人化防止）
5. 週次の夫婦スケジュール共有
