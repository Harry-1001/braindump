# 意思決定・認知負荷・メンタルロード

## メンタルロードとは

「家事・育児における、実際に手を動かす前段階にある『考える・気づく・計画する』負担」。

目に見えないため「invisible load（見えない負担）」とも呼ばれる。

具体的には：
- 子どもの次の受診日の管理
- 食材の在庫確認・献立決定
- 学校行事の事前確認と準備
- 子どもの体調変化のモニタリング
- 習い事の送迎スケジュール調整
- 家族全員の衣類・持ち物管理

## 誰が担っているか

多くの場合、母親が担う構造的傾向がある。理由：

- 「気づき役」になりやすい性格特性（責任感・完璧主義・気遣い）が女性に多く期待される
- 職場での時短・育休取得で「家のことは自分が担う」という自己認識が形成される
- 夫に振っても「管理」自体は妻が担い続けるケースが多い

→ 物理的タスクを渡しても、メンタルロードは移転しない問題

## 疲労が意思決定の質に与える影響

慢性的な疲労状態では：
- 判断力の低下（今夜の夕食メニュー決定すら困難になる「決断疲れ」）
- 感情コントロールの難化 → 子ども・配偶者への八つ当たり
- 長期思考の困難化 → キャリアや人生の設計を後回しにする

## 3人子育てへの影響

子どもが増えるほど、メンタルロードの量は線形以上に増加する：
- 1人：1つのスケジュール系
- 2人：2系統 + 兄弟間調整
- 3人：3系統 + 兄弟間調整 + 年齢差対応（乳幼児 / 幼稚園 / 小学生が混在すると世界が全く異なる）

## 軽減の工夫

1. **見える化**：Googleカレンダーや家族アプリ（TimeTree等）での共有
2. **相談の共有化**：「どうすればいい？」を一人で考えず、夫婦の協議に変える
3. **システム化**：特定曜日に特定タスクを固定するルール化
4. **外注**：献立決定 → ミールキット、掃除 → 家事代行など
