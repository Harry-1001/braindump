# 時間・体力の実態

## 1日のタイムライン（典型例）

| 時間帯 | 内容 |
|--------|------|
| 5:30〜6:30 | 起床・朝食準備・自分の支度 |
| 6:30〜8:00 | 子ども起こし・食事・着替え・登園/登校準備 |
| 8:00〜9:00 | 送り出し・出勤 |
| 9:00〜17:00〜18:00 | 就労 |
| 18:00〜19:00 | 迎え・帰宅 |
| 19:00〜21:00 | 夕食準備・食事・入浴・宿題対応 |
| 21:00〜22:00 | 子どもの就寝対応 |
| 22:00〜24:00 | 家事残業・翌日準備・わずかな個人時間 |

3人の子どもがいる場合、異なる年齢の登園/登校時間・習い事・宿題量が重なり複雑度が増す。

## 睡眠時間の実態

- 2023年国民健康・栄養調査：6時間未満の人が男性38.5%、女性43.6%
- 女性の30代・40代では4割超が睡眠6時間未満
- 睡眠確保の妨げ：30代女性で「育児」が最多回答
- 慢性的な不眠（1ヶ月以上）：成人の約20%（20.6%・2022年調査）

## 3人目の負荷増

- 2人目まで：ルーティン化が可能、対応可能な余裕がある
- 3人目以降：「2人と3人は全然違う」という実感が多数派
- 3人目を持つ親の9/15人（60%）が「3人目の壁を感じた」と回答
- 体力・精神的余裕が限界：2人でいっぱいいっぱいだった家庭が多い

## 自由時間の欠如

子育ての負担感の最多回答：「自分の自由な時間が持てない」42.5%（文科省調査）

夫婦の就業継続のために自由時間は最も削られる。自分の時間は深夜のみとなりやすく、それが睡眠不足に直結する。
