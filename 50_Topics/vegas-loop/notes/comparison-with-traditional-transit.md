# Vegas Loop vs 従来型公共交通の比較

## Summary
Vegas Loopは低コスト・柔軟なステーション配置・直行運行という利点がある一方、輸送能力の限界と「公共交通」としての位置づけに議論がある。地下鉄やLRTと根本的に異なるコンセプトであり、補完的な役割が現実的とする見方が強い。

## Notes

### 比較表

| 項目 | Vegas Loop | 地下鉄 | LRT |
|------|-----------|--------|-----|
| 建設コスト/マイル | 約$10M | $200M〜$1B+ | $50M〜$200M |
| 輸送能力/時間 | 6,600人（現在）→90,000人（目標） | 数十万人 | 数万人 |
| 1車両あたり乗客 | 4〜5人（Tesla）/ 16人（将来シャトル） | 1,000人以上 | 100〜200人 |
| ステーション設置の柔軟性 | 駐車スペース2台分から可能 | 大規模な地下空間が必要 | 中規模 |
| 各駅停車の要否 | 不要（直行） | 必要 | 必要 |
| 工期 | 18ヶ月（LVCC規模） | 5〜10年以上 | 3〜7年 |
| 公的資金依存度 | 低い（民間主導） | 高い | 高い |

### Vegas Loopの利点
1. **コスト**：従来の地下鉄の1/10〜1/100のコスト
2. **ステーション柔軟性**：駐車スペース2台分から設置可能。都市中心部、駐車場、住宅地に容易に統合
3. **直行サービス**：全駅停車しないため、乗客に関連する駅のみ停車
4. **民間資金**：政治的・予算的制約から独立
5. **段階的拡張**：区間ごとに順次開業可能
6. **速度**：各駅停車がないため、バス・地下鉄の2倍の実効速度を主張

### Vegas Loopの課題・批判
1. **輸送能力の根本的限界**：1台4〜5人 vs 1編成1,000人以上。高密度需要への対応に疑問
2. **ピーク時の渋滞**：LVCC内で「トンネル渋滞」が観察されたとの報告
3. **カバレッジの偏り**：カジノ間接続が中心で、一般住民・労働者の日常的移動手段にはならないとの批判
4. **「公共交通」か？**：交通専門家Jarrett Walkerは、Loop乗車後も「公共交通」とは言えないとの見解
5. **スケーラビリティ未証明**：90,000人/時の目標達成には大量の車両とトンネルが必要
6. **ドライバー依存**：自動運転未実現のため人件費が継続

### 交通専門家の見解（Human Transit / Jarrett Walker）
- 「低速車両シャトルは高容量の多線路メトロとは比較にならない」
- ラスベガスのような低密度都市では一定の合理性があるが、高密度都市での代替にはならない
- ステーション数の増加による利便性向上は認めるが、根本的な容量問題は残る

### ラスベガス固有の文脈
- ラスベガスはストリップ沿いのリゾート間移動が主要需要 → Loopモデルと親和性が高い
- 既存の公共交通が弱い都市 → 「何もない」よりは改善
- カジノリゾートがステーション費用を負担する経済モデルが成立する特殊環境
- Brightline West（LA〜LV高速鉄道）との接続で複合的な交通ハブとしての可能性
