# Vegas Loop 料金・利用者数・輸送能力

## Summary
Vegas Loopの運賃は区間により$4.25〜$12で、コンベンション参加者は無料。累計乗客数は300万人を超え、空港接続完了後は年間500万〜1,000万人の利用を見込む。最終形態では時間あたり最大90,000人の輸送能力を目標とする。

## Notes

### 料金体系
| 区間 | 料金 |
|------|------|
| LVCC内（コンベンション参加者） | 無料 |
| リゾート間・LVCC間 | $4.25 |
| Westgate/Resorts World → 空港 | $12 |
| Fontainebleau発 | 現在無料 |

#### 将来の料金（計画）
- 空港 → LVCC：$10
- Allegiant Stadium → LVCC：$6
- ダウンタウン → LVCC：$5

### 支払い方法
- USD（米ドル）
- Dogecoin（暗号通貨）も受付

### 利用者数
- 累計乗客数：300万人以上（2026年初頭時点）
- 空港コネクター完成後の年間目標：500万〜1,000万人
- 参考：旧モノレールのピーク時（2007年）は年間790万人

### 輸送能力
- 現在の処理能力：時間あたり約6,600人
- 最終目標（104駅完成時）：時間あたり90,000人
- 比較：一般的な地下鉄は1編成で1,000人以上を輸送可能

### 運行時間
- 空港サービス：10AM〜9PM
- チケット購入：lvloop.com
