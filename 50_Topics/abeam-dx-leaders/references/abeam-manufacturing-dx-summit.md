# アビームコンサルティングが語る、データ駆動型変革の要諦

## Summary

製造業DXサミット2024における橘知志氏（執行役員プリンシパル、未来価値創造戦略ユニット長）の講演レポート。製造業におけるデータ駆動型変革の推進方法について、単一業務領域のデータ利活用、全社型データ利活用、マスタープラン策定の3段階で解説。

## Key Points

- 登壇者：橘知志（執行役員プリンシパル、未来価値創造戦略ユニット長）
- 「業務や事業の姿とデータ活用のあり方を同時に考えることが肝要」
- ユースケース駆動型アプローチを推奨（ルール策定やガバナンス強化より先にユースケース検討）
- マスタープランでは「なぜ取り組むのか」等の価値観共有文書化が重要
- 製造業の課題：デジタルツインや工場データと設計開発データの連携実装は少ない

## Relevance

ABeamのDX推進者による外部講演の具体例。橘知志氏の専門知見と、ABeamのデータ駆動型変革に関する方法論が詳述されている。
