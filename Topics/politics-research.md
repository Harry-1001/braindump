## 批判的読解フレームワーク

公約を読むとき、以下の問いを順番に当てはめる。

| 問い | 何を見ているか |
|------|--------------|
| 財源は明示されているか？ | 楽観主義の最大の隠れ場所 |
| 誰が得をして、誰が負担するか？ | ロールズ的「ベールの無知」で考える |
| 数値・期限・責任者はあるか？ | 曖昧な言葉は検証不能 |
| 前回の公約はどうなったか？ | 実績との整合性 |
| 因果関係は正しいか？ | 「〜だから〜になる」のロジックを分解する |
| どんな価値観・人間観が前提か？ | サンデル的な問い。自由主義/共同体主義/功利主義？ |

## 各政党サマリー（参院選2025基準）

### 自由民主党

**キャッチコピー**：「日本を動かす 暮らしを豊かに」

**主な公約**
- GDP1000兆円・国民所得5割増を目標
	- 現状GDP: 600兆円
	- US: 4500兆円
	- Chine: 3000兆円
- 「責任ある積極財政」で政府債務残高のGDP比を着実に低下
- 食料品の消費税免除を「国民会議」で検討（時期・財源は未定）
- 強い経済を実現しつつ財政の持続可能性を確保

**批判的評価**

| 観点 | 評価 |
|------|------|
| 財源の明示 | 「税収の上振れを活用」が多く、恒久財源の根拠が薄い |
| 数値の具体性 | GDP1000兆円は目標年次が不明確。「5割増」の達成経路も不明 |
| 論理の整合性 | 「積極財政」と「債務比率低下」の同時達成は論理的緊張がある |
| 前回実績 | アベノミクスの「3本の矢」のうち構造改革は未達のまま引き継がれた |
| 価値観の前提 | 成長主義・国家安全保障を重視。分配より拡大路線 |

**問い**：「責任ある積極財政」の「責任ある」とは具体的に何を指すのか？

### 立憲民主党

**キャッチコピー**：「政権交代こそ、最大の政治改革」

**主な公約**
- 食料品の消費税を1年間ゼロ（2026年4月〜）
- 「食卓おうえん給付金」1人2万円
- 給付付き税額控除の導入（中低所得層への直接支援）
- ガソリン暫定税率廃止（-25円/L）
- 防衛増税なし
- 辺野古移設工事中止
- 赤字国債には頼らない方針を明言

**批判的評価**

| 観点 | 評価 |
|------|------|
| 財源の明示 | 食料品消費税ゼロの財源は「1年間限定」で逃げている。恒久化した場合の試算なし |
| 数値の具体性 | 給付額・期間は具体的。ただし「1年後どうなるか」は不明 |
| 論理の整合性 | 「赤字国債ゼロ」を掲げながら大規模な減税・給付を同時提案。矛盾に近い |
| GX投資目標 | 10年200兆円・年250万人雇用創出は楽観的。雇用創出の因果が不透明 |
| 価値観の前提 | 立憲主義・財政規律・分配重視。保守的改革路線に近い |

**問い**：「給付付き税額控除」は制度設計が複雑で行政コストが高い。実施能力の裏付けはあるか？

### 日本維新の会

**キャッチコピー**：「改革」「副首都」「教育無償化」

**主な公約（維新八策2025、494政策）**
- 社会保険料を年6万円引き下げ（医療費削減を財源に）
- 高齢者の医療費窓口負担を原則3割に見直し
- 教育・保育の無償化と質の向上
- 副首都の設置・中央省庁移転
- 一院制の導入議論
- 食料品消費税を2年間ゼロ（時限措置）
- 防衛費GDP比2%増（増税なし）

**批判的評価**

| 観点 | 評価 |
|------|------|
| 財源の明示 | 「医療費削減で社会保険料引き下げ」は論理的だが、削減額の試算根拠が不透明 |
| 高齢者負担増 | 窓口3割負担は現役世代の受益だが、受益者負担の論理は一定の合理性がある。ただし低所得高齢者への影響が未考慮 |
| 数値の具体性 | 494政策と多いが、優先順位と実施時期が見えない |
| 副首都 | 「首都機能移転」は過去にも議論されて実現していない。コスト試算がない |
| 一院制 | 立法府の代表性の観点から根本的な問いがある。二院制の何が問題なのかの説明不足 |
| 価値観の前提 | 効率・競争・行政スリム化。新自由主義的傾向が強い |

**問い**：「改革」という言葉は維新の最大の武器だが、「何をやめるか」「誰が損をするか」が曖昧にされていないか？

### 国民民主党

**キャッチコピー**：「手取りを増やす夏」

**主な公約**
- 基礎控除引き上げ：103万円の壁を178万円へ（所得税減税）
- ガソリン暫定税率廃止・再エネ賦課金廃止
- 消費税5%への恒久引き下げ
- 奨学金債務減免（最大150万円）
- 「子ども・子育て支援金」（通称「独身税」）廃止
- 増税・社会保険料アップは行わない

**財源論争の焦点：178万円への引き上げ**

| 論点 | 内容 |
|------|------|
| 財務省試算 | 全面実施で国・地方合計約**7.6兆円**の税収減（消費税3%分相当） |
| 実際に合意された措置 | 段階的引き上げで追加減税は年間約6,500億円（財務省試算） |
| 国民民主の反論 | 税収上振れ（2023年度+2.5兆円）・予算不用額（2023年度6.9兆円）を活用できる |
| 批判 | 税収上振れは恒久財源ではない。コロナ特需後は縮小傾向 |

**批判的評価**

| 観点 | 評価 |
|------|------|
| 財源の明示 | 「税収上振れ」を恒久財源とするのは論理的に危うい。景気悪化時に逆回転するリスクあり |
| 数値の具体性 | 178万円・7.6兆円減収など数値は具体的。透明性は高い方 |
| 論理の整合性 | 「増税も社会保険料増もしない」かつ「大規模減税」は財政悪化を前提にしているが、それを明言していない |
| 受益者の偏り | 所得控除の拡大は**高所得者ほど恩恵が大きい**（累進税率のため）。「手取りを増やす」が誰の手取りかを問うべき |
| 価値観の前提 | 個人の可処分所得の最大化を善とする。再分配より自助を重視 |

**問い**：「103万円の壁」は本当に労働参加の壁か？壁を引き上げるより社会保険料の設計を変える方が効果的では？

### れいわ新選組

**キャッチコピー**：「財源はある！」

**主な公約**
- 消費税の廃止
- 一律10万円の現金給付
- 法人税引き上げ・所得税累進性の強化
- 最低賃金1500円（全国一律）
- グリーン産業に10年間200兆円投資（国費5兆円/年）
- 原発即時廃止
- 43兆円の防衛費増額計画の中止

**財政思想の前提**
- 政府は通貨発行権を持つため財政赤字は問題ない（MMT的立場）
- プライマリーバランス黒字化目標を破棄
- 財源は「当面、国債発行で賄う」

**批判的評価**

| 観点 | 評価 |
|------|------|
| 財源の明示 | MMT（現代貨幣理論）を前提としているが、この理論は経済学者の間でも論争的。「インフレになれば増税で調整」という前提が公約に明示されていない |
| 消費税廃止の規模 | 消費税収は年間約20〜23兆円。代替財源として法人税・富裕層課税だけでは足りない可能性が高い |
| 最低賃金1500円 | 地方の中小企業への影響試算が必要。急激な引き上げは雇用減少リスクもある |
| 10万円給付 | 全国民への一律給付は高所得者にも届く。ターゲティングの効率が悪い |
| 価値観の前提 | 反緊縮・再分配重視・弱者保護。共同体主義・福祉国家志向 |

**問い**：「財源はある」という主張の論拠はMMT理論だが、日本でインフレが起きた場合の「増税によるブレーキ」を実際に踏めるのか？政治的実現可能性は？

### 公明党

**キャッチコピー**：「訴えるだけじゃ変わらない。公明党は実現する。」

**主な公約**
- 食料品の消費税率引き下げ（自民と調整中）
- 1人あたり年2〜4万円の所得税減税
- 18歳以下に一人4万円の還元
- 自動車税制の抜本見直し
- 社会保障の充実

**批判的評価**

| 観点 | 評価 |
|------|------|
| 財源の明示 | 減税・給付の財源が不明。「実現する」と言うが、自民との調整次第で骨抜きになりやすい |
| 自民との関係 | 食料品減税を訴えるが自民が強く反対。連立の制約の中でどこまで実現できるか疑問 |
| 独自性 | 政策の多くが与党調整の産物。独立した哲学が見えにくい |
| 気候変動 | 気候ネットワークの評価で最低水準。温室効果ガス削減目標を明示していない |
| 価値観の前提 | 生活者・生命重視。宗教的背景からくる人道主義。ただし政策への反映は限定的 |

## 横断的論点：政党を超えて問うべきこと

### 1. 「手取りを増やす」競争の構造的問題
ほぼ全政党が「減税・給付」を訴えている。誰も「どこを削るか」を明言していない。これは有権者への「正直さ」という観点から問題がある。

### 2. 財源の「楽観的前提」パターン
| パターン | 使っている党 |
|----------|------------|
| 税収上振れを恒久財源とする | 国民民主、自民 |
| 国債発行で当面賄う | れいわ |
| 歳出削減で賄う（具体性なし） | 維新 |
| 財源を「検討する」 | 公明、自民（食料品減税） |

### 3. サンデル的な問い：「何が善い社会か」を各党はどう考えているか
| 政党 | 暗黙の社会観 |
|------|------------|
| 自民 | 成長した国が個人を守る（国家主義的成長主義） |
| 立憲 | 法の支配のもとで権利を守る（リベラル立憲主義） |
| 維新 | 効率的な行政が個人の自由を拡張する（新自由主義） |
| 国民民主 | 個人の可処分所得が最大化される社会（経済的個人主義） |
| れいわ | 弱者が排除されない社会（再分配的共同体主義） |
| 公明 | 生命と生活が守られる社会（人道主義） |

### 4. 検証されていない因果関係の例
- 「減税→消費拡大→税収増」（トリクルダウン型。過去の実証は限定的）
- 「社会保険料削減→可処分所得増→少子化解消」（少子化の原因は多因子）
- 「防衛費増額→抑止力向上→安全」（抑止理論は複雑で一対一対応ではない）
