# 川名麻耶

[[孫正義]]（ソフトバンク創業者）の長女。1981年生まれ。慶應義塾一貫教育を経てゴールドマン・サックスに入行後、実業家へ転身。2019年にBOLD設立、2026年4月にはスパイバーのCEOに就任。2025年末まで孫正義の娘であることを公表していなかった「沈黙の戦略家」。

> 注意：ユーザーリクエストでは「孫正義の妻」と書かれていたが、川名麻耶は孫正義の**長女**。孫正義の妻（川名麻耶の母）は[[大野優美]]。

### 基本情報
- 生年：1981年
- 父：[[孫正義]]（ソフトバンクグループ会長兼社長）
- 母：[[大野優美]]（専業主婦、孫正義と1979年学生結婚）
- 3人の子供の母
- 夫：非公開（実業家とされる）

### 教育
- 慶應義塾幼稚舎 → 中等部 → 高等部 → 慶應義塾大学経済学部（16年間一貫、推定2,000万円超の教育投資）
- 大学では国際貿易論ゼミとテニスサークルの代表を兼任

### 出自公表の戦略的意義
2025年末のスパイバー支援発表に際して初めて「孫正義の娘」と明かした。短期的キャピタルゲインを求めない長期支援者としての信頼性を示すためとされる。一部の財界・投資家の間ではすでに知られていた存在だったが、それ以外の場では意図的に秘匿してきた。
