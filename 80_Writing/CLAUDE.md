# Writing — note.com Articles

## Structure
```
Writing/
  drafts/     # 執筆中・下書き
  published/  # 公開済み
```

## Frontmatter（必須）
```yaml
---
status: draft | published
genre: [life, romance, philosophy, business, travel]
tags: [genre/life, genre/romance, ...]
---
```

## Genre Tags
- `#genre/life` — 日常・体験・自己
- `#genre/romance` — 恋愛・パートナー
- `#genre/philosophy` — 哲学・自己探求
- `#genre/business` — ビジネス・仕事
- `#genre/travel` — 旅

## Rules
- 公開したら drafts/ → published/ に移動してstatusをpublishedに変更
- 原文は書き換えない（加筆はOK）
