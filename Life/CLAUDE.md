# Life — Personal Notes

## Structure
```
Life/
  profile/
    me.md       # 性格・価値観・気質
    goals.md    # 目標・目指す先
    history.md  # 経歴・人生のサマリー
  日記/          # Daily journal entries (YYYYMMDD.md)
```

## Journal (日記)
- Filename format: `YYYYMMDD.md`
- Free-form, no strict structure required
- Frontmatter optional

## Writing (日記・メモ) のルール
- 内容をそのまま残す（書き換えない）
- タグ・wikilinksの追加は/dumpで行う
- 感情・体験をそのまま書いてOK

## Profile Files
- `me.md` `goals.md` `history.md` は定期的に更新する
- これらはClaude Codeのグローバル設定（~/.claude/CLAUDE.md）からも参照される
